## About this analysis

This intentionally fictional example calculates one storage bin from 0.8 kg
of mock polypropylene granulate. The foreground graph references only one
background activity, while that activity consumes mock grid electricity.

It is the smallest useful example for testing background activity resolution,
LCIA results, background Sankey nodes, and REST/MCP compatibility. Do not use
its invented inventory values for environmental claims.

Using EF v3.1, the expected climate-change result is approximately
**1.440000 kg CO2-Eq**. The 0.8 kg resin demand includes its fictional direct
emissions and the emissions from 1.6 kWh of upstream mock electricity.
